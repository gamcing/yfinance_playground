# 以金融为例子学习Python的一个简单教程

## 环境准备
本项目使用uv管理python版本，可以使用pip安装uv：

```bash
pip install uv
```

## 安装完成后，使用uv安装项目依赖：

```bash
uv sync
```

## 项目主程序

[jupyter_handbook.ipynb](jupyter_handbook.ipynb) 是一个jupyter notebook文件，包含了金融数据分析的示例代码和解释。

你可以使用VS Code运行这个notebook，或者使用Jupyter Notebook / JupyterLab来打开它。
